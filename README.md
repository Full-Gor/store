# Store

Plateforme de déploiement d'applications mobiles.

## Description

Store est une plateforme permettant de déployer, distribuer et gérer des applications mobiles de manière simple et efficace.

## Fonctionnalités

- Déploiement d'applications iOS et Android
- Gestion des versions et des releases
- Distribution aux testeurs et utilisateurs
- Tableau de bord de suivi des installations

## Installation

```bash
# Cloner le dépôt
git clone git@github.com:Full-Gor/store.git

# Accéder au répertoire
cd store
```
